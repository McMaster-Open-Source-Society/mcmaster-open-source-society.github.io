[ moss/home ]

Technical work. Public repos. Student builders.

MOSS is a student-run coding club for collaborative software development, open source contribution, and practical project experience at McMaster.

Whether you are learning Git for the first time or already maintaining repositories, MOSS gives you a place to contribute consistently, learn from peers, and publish work that matters.

[ mission ]

What MOSS is built for

Open source onboarding

Learn Git, GitHub, pull requests, code review, and repository workflow in a practical setting.

Project-based experience

Contribute to visible repositories that strengthen portfolios, resumes, and technical confidence.

Flexible involvement

Participate through events, project work, or async contribution without heavy club overhead.
